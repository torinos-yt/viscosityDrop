## GUIパラメータ  
- RENDER : 最終レンダリングを表示  
- CIRCLE : フィルタ前の状態を表示  
- BLURRED : ぼかしフィルタがかかった状態を表示
- BLURWIDTHSLIDER : ぼかしの幅を調整するスライダー
- BLURPASSSLIDER : ぼかしのパス数（フィルタをかける回数）を調整するスライダー
- THRESHOLDSLIDER : 2値化の際の閾値を調整するスライダー
- THRESHWIDSLIDER : 2値化のスムージングの幅を調整するスライダー
